Context: The detect() method above returns std::optional<LoopCandidate>. This is cleaner than returning a bool + out-parameter, or using a sentinel value. The caller checks with has_value() or uses value_or(). std::optional avoids heap allocation (unlike std::unique_ptr) and communicates intent clearly: "this function might not produce a result."
