Context: Sophus is a header-only C++ library that provides Lie group types (SO3, SE3) built on Eigen. Unlike using raw 4×4 matrices, Sophus::SE3d ensures that transformations always remain on the SE(3) manifold (valid rotation + translation). Composing poses is simply multiplication: T_world_cur = T_world_kf * T_kf_cur. Sophus also provides log/exp maps needed by optimisers (e.g., Ceres) to parameterise incremental updates in the tangent space.
