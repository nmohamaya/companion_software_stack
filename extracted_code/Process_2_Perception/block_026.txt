  TrackedObjectList (2D)    LiDARClusters (3D)    RadarDetections (range+vel)
       │                         │                        │
       ▼                         ▼                        ▼
  ┌──────────────────────────────────────────────────────────────┐
  │ STEP 1: PROJECT LiDAR → CAMERA                               │
  │                                                                │
  │ For each LiDAR cluster centroid [X,Y,Z] in 3D:               │
  │   [u,v] = K × [R|t] × [X,Y,Z,1]^T                           │
  │   (K = camera intrinsic, [R|t] = LiDAR→Camera extrinsic)     │
  │   → Now we know WHERE each cluster appears in the image       │
  └──────────────────────────────┬───────────────────────────────┘
                                  ▼
  ┌──────────────────────────────────────────────────────────────┐
  │ STEP 2: ASSOCIATE TRACKS ↔ LiDAR CLUSTERS                    │
  │                                                                │
  │ For each tracked object's 2D bbox:                            │
  │   Find LiDAR cluster whose projected centroid falls inside    │
  │   the bbox (or closest within threshold)                       │
  │   Match → object gets 3D position from LiDAR centroid         │
  │   No match → object remains 2D-only (camera estimate)         │
  └──────────────────────────────┬───────────────────────────────┘
                                  ▼
  ┌──────────────────────────────────────────────────────────────┐
  │ STEP 3: CROSS-CHECK WITH RADAR                               │
  │                                                                │
  │ For each 3D-fused object:                                      │
  │   Find radar detection within:                                │
  │     ± 2m range AND ± 15° azimuth                              │
  │   Match → object gets Doppler velocity (direct measurement)   │
  │   No match → object velocity from Kalman estimate only        │
  │   Radar-only → create low-confidence "radar ghost" object     │
  └──────────────────────────────┬───────────────────────────────┘
                                  ▼
  Output: FusedObjectList (track_id, class, 3D pos, vel, confidence)
          → Shared Data Bus → Mission Planner, Comms
