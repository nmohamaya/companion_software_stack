  └──────────┬────────────────┘
             ▼
  ┌───────────────────────────┐
  │ 3. POST-PROCESSING        │
  │    Decode bbox predictions │  Convert from grid offsets to pixel coords
  │    Confidence threshold   │  Keep only conf > 0.5
  │    NMS (Non-Max Suppress) │  Remove overlapping boxes (IoU > 0.45)
